Comment calculer une recette pour plus ou moins de personnes ?

Adapter les quantites d'une recette est essentiel quand vous cuisinez pour un nombre different de convives. Notre calculateur de recette gratuit vous permet de :

  • Calculer automatiquement les nouvelles quantites selon le nombre de portions
  • Utiliser un tableau de proportionnalite pour vos recettes de cuisine
  • Effectuer le calcul des quantites en patisserie avec precision
  • Adapter les recettes de Thermomix, Companion ou Monsieur Cuisine

Calcul recette : la formule simple

Pour adapter une recette, divisez chaque quantite par le nombre de portions original, puis multipliez par le nombre souhaite. Notre outil fait ce calcul automatiquement pour vous !
