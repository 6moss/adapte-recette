Comment copier la description ?

Instagram :

  1. Ouvrez le Reel dans l'app
  2. Appuyez sur "... plus" sous la description
  3. Maintenez appuye sur le texte et selectionnez "Copier"

TikTok :

  1. Ouvrez la video dans l'app
  2. Appuyez sur la description pour la voir en entier
  3. Maintenez appuye et copiez le texte
