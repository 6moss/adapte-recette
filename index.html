Instagram :

  2. Appuyez sur la description pour la voir en entier

Comment calculer une recette pour plus ou moins de personnes ?

Adapter les quantités d'une recette est essentiel quand vous cuisinez pour un nombre différent de convives. Notre calculateur de recette gratuit vous permet de :

  • Calculer automatiquement les nouvelles quantités selon le nombre de portions
  • Utiliser un tableau de proportionnalité pour vos recettes de cuisine
  • Effectuer le calcul des quantités en pâtisserie avec précision
  • Adapter les recettes de Thermomix, Companion ou Monsieur Cuisine

Calcul recette : la formule simple

Pour adapter une recette, divisez chaque quantité par le nombre de portions original, puis multipliez par le nombre souhaité. Notre outil fait ce calcul automatiquement pour vous !
